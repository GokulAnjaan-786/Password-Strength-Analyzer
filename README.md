# 🔐 Password Strength Analyzer

A cybersecurity-based Python project developed in Kali Linux that analyzes the strength of user passwords and provides security suggestions to create stronger passwords.

This project helps users understand password security, password complexity, hashing, and secure authentication concepts in a simple way.

---

# 🚀 Features

- ✅ Checks password length
- ✅ Detects uppercase and lowercase letters
- ✅ Detects numbers and special symbols
- ✅ Calculates password strength score
- ✅ Gives password improvement suggestions
- ✅ Generates SHA-256 password hash
- ✅ Detects weak/common passwords
- ✅ Beginner-friendly cybersecurity project

# ⚙️ How It Works

1. User enters a password
2. System checks:
   - Length
   - Uppercase letters
   - Lowercase letters
   - Numbers
   - Special characters
3. Password score is calculated
4. Strength level is displayed
5. Suggestions are provided
6. SHA-256 hash is generated
